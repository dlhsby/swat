# FAQ & Bantuan

### Menu yang saya butuhkan tidak muncul di sidebar

Menu mengikuti **hak akses** akun. Bila sebuah modul tidak tampil, peran Anda belum
diberi izin untuk itu. Hubungi administrator — lihat [Peran & Hak
Akses](/memulai/peran-akses).

### Saya lupa kata sandi

Gunakan tautan **Lupa kata sandi** di halaman masuk, atau minta administrator menyetel
ulang kata sandi Anda.

### Saya diminta mengganti kata sandi saat masuk

Itu normal untuk akun baru: penggantian kata sandi **wajib** pada login pertama demi
keamanan. Setelah diganti, Anda langsung masuk ke dasbor.

### Bagaimana mengganti bahasa atau tema?

Buka menu avatar (kanan atas) → **Pengaturan**. Di sana Anda dapat memilih tampilan
(Sistem / Terang / Gelap) dan bahasa (Bahasa Indonesia / English).

### Data yang saya lihat tampak tidak mutakhir

Muat ulang halaman. Bila masih janggal, catat langkah yang Anda lakukan dan laporkan ke
administrator/tim teknis.

### Istilah di aplikasi

- **Pengangkutan Sampah** — satu kegiatan angkut oleh armada (istilah teknis: _Haul_).
- **Perjalanan** — satu trayek/putaran dalam pengangkutan (istilah teknis: _Trip_).
- **Retribusi** — pungutan atas layanan (istilah teknis: _Levy_).
- **Jatah Kitir** — izin/kuota pembuangan di TPA.
