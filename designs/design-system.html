Sistem Desain · v1.0

SWAT Design System

Fondasi visual untuk aplikasi back-office pengangkutan sampah DLH Kota Surabaya. Token, tata letak, dan komponen — dipetakan 1:1 ke Tailwind CSS + shadcn/ui, antarmuka Bahasa Indonesia, dan memenuhi WCAG 2.1 AA.

Tema terang (baseline) Desktop-first Plus Jakarta Sans Klik swatch mana pun untuk menyalin token
01

Jelas, bukan dekoratif

Tanpa gradien atau ornamen. UI yang berbicara sendiri.

02

Padat, tetap lega

Grid 8px, 25–100 baris per tabel tanpa scroll di 1280px+.

03

Cepat lewat keyboard

Tab order rapi, fokus terlihat, entri angka langsung.

04

Cegah kesalahan

Validasi inline, konfirmasi aksi merusak, pesan manusiawi.

Fondasi

Warna

Hijau emerald untuk identitas DLH/lingkungan; slate netral untuk teks dan permukaan; warna semantik standar Tailwind. Permukaan hijau dengan teks putih memakai primary-700 ke atas agar lolos kontras AA.

Aturan kontras

Isian hijau + teks putih → primary-700 (5,49:1 ✓). primary-600 dipakai untuk ring fokus, ikon, dan border 2px — bukan teks di atas putih (3,78:1 ✗).

green-700 ✓

Fondasi

Status & Enum Domain

Setiap nilai enum dipetakan 1:1 ke satu pill — latar -100, teks -700, semua diverifikasi ≥4,5:1. Bahasa: amber = berjalan/menunggu · biru = selesai · hijau = terverifikasi/baik · merah = rusak kritis · slate = nonaktif/hilang. Warna selalu disertai teks & titik.

⚠Anda sedang offline — perubahan disinkronkan saat daring
  • Service worker meng-cache app-shell (Fase 1); API network-first.
  • Tombol & bar menghormati env(safe-area-inset-*) (notch / home bar).
