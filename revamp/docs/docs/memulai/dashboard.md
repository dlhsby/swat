# Mengenal Dasbor

Setelah masuk, Anda tiba di **Dasbor** — ringkasan kondisi terkini pengangkutan sampah
dan retribusi.

![Dasbor SWAT](/img/web/dashboard.png)

## Bagian-bagian layar

- **Bilah atas (topbar)** — pencarian, notifikasi, dan menu akun (avatar) untuk
  membuka **Pengaturan** (tema terang/gelap & bahasa) serta keluar.
- **Sidebar kiri** — navigasi utama, dikelompokkan menjadi Pemantauan, Data Master,
  Pengangkutan, dan Pengguna & Akses. Klik ikon lipat untuk merampingkan sidebar
  menjadi rel ikon.
- **Area konten** — kartu ringkasan dan grafik kinerja hari/periode berjalan.

## Berpindah halaman

Gunakan sidebar untuk membuka setiap modul. Setiap kelompok dapat dibuka/ditutup.
Halaman yang belum tersedia untuk peran Anda tidak ditampilkan.

:::tip Ganti tema & bahasa
Buka menu avatar → **Pengaturan** untuk beralih tampilan (Sistem / Terang / Gelap) dan
bahasa (Bahasa Indonesia / English). Pilihan tersimpan di peramban Anda.
:::
